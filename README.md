# The Calculator

The fifth project of **[The Odin Project](https://www.theodinproject.com/)** Online Calculator web app with a basic calculator operations that also saves calculations on the web app.
You can try it **[here](https://bishomilad.github.io/The-calculator/)**

## Features

- Working basic calculator
- Saving every calculation in the paper section of the web app
- Supporting floating points
- "All clear" button feature that resets the calculator
- "clear" button feature that deletes only the last digit
- Easy to navigate UI
- Clear History feature (either a whole wipe or individual calculation);
